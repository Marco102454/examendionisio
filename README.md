# 🌐 Blog Estático con Filtros por Categoría

Este proyecto es un blog estático construido con HTML, CSS y JavaScript. Incluye un sistema de filtros por categoría para facilitar la navegación entre las publicaciones.

---

## 🚀 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (vanilla)
- Git & GitHub Pages (para publicación)

---

## 📋 Actividades Prioritarias del Proyecto

1. **Definir el objetivo y público del blog**  
   Establecer el propósito y los temas que abordará el blog.

2. **Estructurar el sitio**  
   Crear secciones como: Inicio, Entradas, Categorías, Sobre mí y Contacto.

3. **Diseñar la estructura de carpetas y archivos**  
   Organizar carpetas: `/css`, `/js`, etc.

4. **Desarrollar el maquetado HTML**  
   Crear las páginas con estructura semántica.

5. **Estilizar con CSS**  
   Aplicar diseño visual y hacerlo responsivo para móviles y tablets.

6. **Agregar entradas de blog**  
   Incluir contenido de prueba (título, Contenido, categoría, texto).

7. **Implementar filtros por categoría (JavaScript)**  
   Programar funcionalidad para mostrar u ocultar entradas según su categoría.

8. **Realizar pruebas en navegadores y dispositivos**  
   Validar diseño responsivo, navegación y filtros.

9. **Optimizar y corregir errores**  
   Revisar código, limpiar estilos innecesarios y asegurar accesibilidad mínima.

10. **Publicar el blog**  
    Subir a GitHub Pages, Netlify o plataforma similar.
